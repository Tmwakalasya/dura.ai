## Install

```bash
pip install dura
```

Zero dependencies. Python 3.9+.

## Wrap your agent

```python
from dura import durable

@durable
def book_trip(ctx, dest, amount):
    # Step 1 — charge the card. If we crash after this, it won't fire again.
    charge = ctx.step(
        "charge_card",
        lambda: charge_card(amount),
        undo=lambda: refund_card(charge["id"]),  # auto-refund if later step fails
    )

    # Step 2 — if this fails, charge_card is automatically refunded.
    flight = ctx.step("book_flight", lambda: book_flight(dest))

    # Step 3 — safe to retry; only runs if both above succeeded.
    ctx.step("update_crm", lambda: update_crm(flight))

    return flight
```

That's the entire API. Every `ctx.step` is:

- **Exactly-once** — if the process crashes and restarts, completed steps are skipped.
- **Rollback-able** — if a later step fails, completed steps undo in reverse order.
- **Durable** — the return value is committed to a write-ahead log before the workflow continues.

## Run it

```python
# First run — all three steps execute.
book_trip("Tokyo", 412)

# Process crashes here. Restart with the same arguments:
book_trip("Tokyo", 412)
# ↑ charge_card is SKIPPED — already committed to the log.
```

dura derives a run ID from the function name and inputs (SHA-256), so re-calling
with the same arguments automatically resumes from the last checkpoint. No
explicit run ID or infrastructure changes needed.

## Verify it works

```bash
pip install dura
python -m pytest -q    # 17 passing, 0 xfail
```

Or run the crash/resume demo yourself:

```bash
python examples/book_trip.py crash   # crashes after charging the card
python examples/book_trip.py         # restarts — charge is skipped
```
